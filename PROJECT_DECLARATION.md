## PROJECT SCOPE DECLARATION

### Core Features Developed:
1. AI-powered blog content generation using Google Gemini AI
2. Rich text editor for content creation and editing (Quill)
3. Image upload with automatic CDN optimization (ImageKit)
4. User comment system with admin moderation workflow
5. JWT-based admin authentication and authorization
6. Blog categorization and organization system
7. Publish/Draft management system
8. Admin dashboard with real-time statistics
9. Responsive web design for all devices
10. RESTful API architecture

### Technologies Used:

**Frontend Stack:**
- React 18.3.1 - User interface library
- Vite 5.4.10 - Build tool and development server
- React Router 6.27.0 - Client-side routing
- Quill 2.0.2 - Rich text editor
- Axios 1.7.7 - HTTP client
- React Hot Toast 2.4.1 - Notification system
- Marked 12.0.2 - Markdown parser
- Tailwind CSS 3.4.14 - Styling framework

**Backend Stack:**
- Node.js 18+ - JavaScript runtime
- Express.js 5.0.1 - Web application framework
- MongoDB - NoSQL database
- Mongoose 8.8.1 - Object Data Modeling (ODM)
- JSON Web Token (JWT) 9.0.2 - Authentication
- Multer 2.0.0-rc.4 - File upload middleware
- CORS 2.8.5 - Cross-Origin Resource Sharing
- dotenv 16.4.5 - Environment variable management

**External Services:**
- Google Gemini AI (gemini-2.0-flash-exp) - Content generation
- ImageKit - Image CDN and optimization
- MongoDB Atlas - Cloud database hosting

**Development Tools:**
- Git - Version control
- GitHub - Repository hosting
- Visual Studio Code - Code editor
- Postman - API testing
- ESLint - Code quality
